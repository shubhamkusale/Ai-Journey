## 🧠 How I Learn

I use a **Teach → Test → Recall loop** every single day:

```
1. See a code pattern (5 min)
2. Understand the logic (5 min)  
3. Close everything and recreate from memory (10 min)
4. Fix what breaks (learn from failure)
5. 15-min solo practice every night in VS Code
```

> The blank page was my biggest enemy. This system is how I defeated it.
