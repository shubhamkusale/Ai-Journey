# Week 9 — Hyperparameter Tuning & Unsupervised Learning

## What I Built
- Email Spam Classifier deployed live on Streamlit (99.2% accuracy)
- Customer Segmentation using K-Means — found 5 customer groups
- Hyperparameter tuning with GridSearchCV to find best model settings

## Key Concepts Learned
- GridSearchCV: tests every parameter combination to find best accuracy
- Cross Validation: tests model on multiple data splits for trustworthy results
- K-Means Clustering: groups unlabeled data by similarity
- Elbow Method: finds optimal number of clusters (K=5 for mall customers)
- StandardScaler: scales features to mean=0, std=1 for distance-based models

## Results
- Spam Classifier: 99.2% (live demo deployed)
- Cancer Model GridSearch: best params max_depth=7, n_estimators=200
- Customer Segments: 5 distinct groups profiled and named
